# FounderOS

> **Your AI Chief of Staff for Startup Operations**  
> *Plan. Execute. Track. Ship.*

FounderOS is an AI-powered operating system that helps startup founders execute their entire business workflow—from idea to launch—without hiring a large operations team. It is built with a **retro terminal + futuristic dashboard + neon blue theme** inspired by hackathon aesthetics.

---

## 🚀 Key Features

* **Module 1: Dashboard** - Displays your active startup mission progress, standup updates, and AI recommendations.
* **Module 2: Goal Creator** - Converts a simple startup idea into a full Product Roadmap and Milestones through an interactive Q&A wizard.
* **Module 3: AI Planner** - Shows a week-by-week sprint timeline with timeline optimization controls.
* **Module 4: Task Board** - A Kanban board featuring micro-task breakdown capabilities powered by an AI expander.
* **Module 5: Document Center** - Synthesizes Product Requirements Documents (PRDs), Pitch Decks, and Launch Checklists.
* **Module 6: Marketing Assistant** - Generates copy assets tailored for Twitter/X, LinkedIn, Reddit, and launch emails.
* **Module 7: Team Management** - Identifies blockers and deploys Slack Resolver Agents to resolve dependencies.
* **Module 8: Analytics Dashboard** - Renders completion gauges and metrics forecasting deadline confidence.
* **Module 9: AI Risk Detector** - Scans active systems using a radar interface to highlight potential delays and mitigations.
* **Module 10: AI Agents Console** - Manages specialized agents (Product Manager, Project Manager, Marketing, Operations, Documentation, Analytics).

---

## 🛠️ Tech Stack

* **Frontend**: Pure HTML5 (Semantic Structure) + Vanilla CSS (Custom properties, keyframe animations, glowing layouts) + Vanilla JavaScript (Reactive state, routing, typewriter simulations).
* **Integrations**: Inspired by the Lemma SDK (Agents, Datastores, Workflows, Documents, and Functions).

---

## ⚡ How to Run

No servers, compilers, or build steps are required. 

1. Clone or navigate to the repository directory.
2. Open `index.html` directly in any modern web browser.
3. Access the Command Palette anywhere on the screen by pressing `Ctrl + K`.
